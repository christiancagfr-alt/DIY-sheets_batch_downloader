# DIY下载器 v1.1.0

## 本版亮点

- 全新应用 Logo / 图标
- 新增 **YouTube / Facebook 视频批量下载** 独立板块
  - 单视频 / 播放列表自动加载（无需先解析）
  - 断点续传、批量多链接
  - 画质选择、按平台/播放列表分文件夹
- 新增 **自动检查更新**
  - 启动时静默检查 GitHub Release
  - 可手动点「检查更新」
  - 下载后询问是否安装/替换并重启
- Windows 提供两种产物：
  - `DIYDownloader-v*-windows-setup.exe` **可安装版**（推荐）
  - `DIYDownloader-v*-windows.exe` 便携版

## 使用说明

1. 推荐下载 **windows-setup** 安装包，安装后从开始菜单或桌面启动。
2. 打开软件后可切换「表格 / 粘贴链接」与「YouTube / FB 视频」两个标签。
3. 新版本发布后，软件会提示更新；也可点击右上角「检查更新」。

## 安全说明

请勿将 `token.json`、Google 服务账号等凭据文件上传到仓库或分享给他人。
